System Requirements (系统要求)

Modified on: Wed, 4 Sep, 2019 at 2:38 PM

Note: Advice in this article will only work for JxBrowser 6. See the corresponding article for JxBrowser 7 here.(注意：本文中的建议仅适用于JxBrowser6，JxBrowser7相应文章请点击这里。)


Software Requirements(软件需求)

JxBrowser runs on Windows, and macOS platforms (Intel and AMD), both 32- and 64-bit, and Linux 64-bit:(JxBrowser可在Windows和macOS平台（Intel和AMD）（32位和64位以及Linux 64位）上运行：)


Windows (32-bit & 64-bit)(Windows（32位和64位）)

Windows 7(Windows 7的)

Windows 8(Windows 8)

Windows 8.1(Windows 8.1)

Windows 10(Windows 10)

Windows Server 2008 R2(Windows Server 2008 R2)

Windows Server 2012(Windows Server 2012)

Windows Server 2012 R2(Windows Server 2012 R2)

Windows Server 2016(Windows Server 2016)

Linux (64-bit)(Linux（64位）)

Ubuntu 17.04 Desktop(Ubuntu 17.04桌面)

Ubuntu 14.04+(Ubuntu 14.04以上)

Debian 8+(Debian 8+)

Fedora 24+(Fedora 24+)

openSUSE 13.3+(openSUSE 13.3以上)

RedHat Enterprise Linux 7(红帽企业Linux 7)


Note: Chromium doesn't work in a headless environment. In order to use JxBrowser in Linux headless environment, you must start X Server.(注意：铬在无头的环境中不起作用。为了在Linux无头环境中使用JxBrowser，必须启动X Server。)


macOS (64-bit)(macOS（64位）)

macOS 10.10.x - 10.14.x (Core i5, Core i7)(macOS 10.10.x-10.14.x（Core i5，Core i7）)


Note: macOS environment must be in non-headless mode because Chromium doesn't support headless mode.(注意：由于Chromium不支持无头模式，因此macOS环境必须处于非无头模式。)


Android(安卓系统)

JxBrowser doesn't support the Android due to the fact that Android OS uses the Google JVM which is not supported by JxBrowser (see the JRE paragraph).(由于Android操作系统使用JxBrowser不支持的Google JVM，因此JxBrowser不支持Android（请参见JRE段落）。)


iOS(的iOS)

iOS doesn't support any JVM convenient with the JxBrowser library, therefore JxBrowser doesn't support iOS.(iOS不支持JxBrowser库提供的任何方便的JVM，因此JxBrowser不支持iOS。)


JRE (Oracle, IBM, Apple) 32-bit & 64-bit(JRE（Oracle，IBM，Apple）32位和64位)

Swing: OpenJDK 11, Oracle (Sun) JDK 6.x and higher, IBM JDK 7.x and higher (only Windows and Linux), Apple JDK 6.x and higher (only macOS).

JavaFX: Oracle (Sun) JDK 8.x-10.x, OpenJFX 11.x.

Hardware Requirements(硬体需求)

Intel Pentium 4 or later, 512 MB RAM minimum, 350 MB Hard Disk Space.(Intel Pentium 4或更高版本，最低512 MB RAM，350 MB硬盘空间。)


Content scaling, HiDPI Monitors & Retina(内容缩放，HiDPI监视器和视网膜)

JxBrowser supports the content scaling on both Windows and macOS platforms with Retina displays.(JxBrowser支持带有Retina显示屏的Windows和macOS平台上的内容缩放。)


ARM(臂)

JxBrowser doesn't support ARM architecture right now.(JxBrowser目前不支持ARM体系结构。)
